# IDEA创建maven项目缺少pom.xml

创建maven项目提示pom.xml' already exists in VFS

原先删除掉Maven工程时，只删除了对应的项目文件夹，还得删除C:\Users\Administrator\\.IntelliJIdea日期\config\tasks目录下对应原先Maven工程名字的两个压缩包。
